# TT-260 — Iterative Counterjump Steering (ICS)

## Thesis
If intervention must be minimal and authenticated, steering must be iterative.

## Logic
Large edits amplify divergence. Small verified deltas preserve continuity.
Each iteration re-evaluates state within bounded cancellation windows.

ICS is a control abstraction, not a physics claim.
