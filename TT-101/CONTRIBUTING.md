# Contributing

Contributions welcome if they improve clarity, structure, or safety.

## Rules
- Keep claims scoped: **conceptual** unless backed by primary sources.
- Avoid sensationalism.
- No instructions for harm or destabilization.
- Prefer append-only updates (new docs, new versions).

## Suggested contributions
- Better diagrams
- Tighter glossary
- More templates for journaling / pivot detection
- A small simulation toy model (non-physics) illustrating attractors in decision systems
