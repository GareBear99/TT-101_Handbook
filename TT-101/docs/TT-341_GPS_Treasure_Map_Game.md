# TT-341 — GPS Treasure Map Game (3 Global Spots)

> “Somewhere, something incredible is waiting to be known.”
> — Carl Sagan

---

**Goal:** learn to resolve coordinates into real-world places, like a continuity “treasure hunt”.

## How to Play
1) Copy each coordinate into a map app.
2) Identify the location.
3) Write the location name and country/region.
4) Optional: zoom out and take a screenshot for your journal.

### Spot 3

## Bonus Challenge (Continuity Skill)
For each spot, answer:
- Why is this location likely to persist in cultural memory?
- What could be left here that a future reader might rediscover?

## Notebook Entry
Create a “Treasure Log”:
- Spot #
- Location name
- Country/region
- One sentence why it matters
