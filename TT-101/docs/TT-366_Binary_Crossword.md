> “Knowledge preserved is civilization extended.”

This segment adds a **crossword-style puzzle** designed to train binary decoding skills used in the **QR slate / optical slate** workflow.

## What this is
- A small crossword grid containing *time-beacon / archive* vocabulary.
- **Clues are 8-bit binary ASCII** (byte strings).
- The puzzle is solvable offline and is included as an interactive handbook page.

## How to play
1. Open the interactive page: `handbook/TT-366.html`
2. Choose a clue.
3. Decode the binary bytes (or use TT-365 Decoder).
4. Type the letters into the crossword grid.
5. Use **Check** to validate your progress.

## Target vocabulary (solution set)
Across:
- SLATE
- HASH
- BINARY
- CONTINUUM

Down:
- SIGNAL
- BEACON
- ARCHIVE
- TIMELINE

## Notes
- Binary is provided in **8-bit groups** for maximum universality.
- The interactive page can optionally reveal ASCII equivalents as a learning aid.
