# Glossary

- **Branch**: A non-interacting macro-history resulting from a divergence (in branching models).
- **Block universe**: A model where past/present/future are all part of a fixed 4D spacetime.
- **Chronology protection**: Hypothesis that physics prevents practical time machines / paradoxical loops.
- **Continuity**: The subjective persistence of identity / narrative through time.
- **Decoherence**: The process by which quantum superpositions become effectively classical alternatives (no interference).
- **Entropy**: Measure of microscopic disorder / number of microstates consistent with a macrostate.
- **Pivot window**: A bounded period where a small intervention produces large life trajectory changes with minimal collateral.
- **Minimal-impact message budget**: A constraint limiting how much future knowledge can be injected without destabilizing history.
- **Handshake protocol**: A proof system to authenticate a claimed continuity agent.
- **Timeproof Continuity Protocol (TPCP)**: The repo’s operational framework for safe hypothetical intervention.

- **Temporal Beacon**: A publicly verifiable pivot marker (timestamped + hashed) intended to be discoverable by hypothetical future observers.
- **Observation Window**: The time interval in which a beacon response would be meaningful; non-response is treated as ambiguous.
- **Listener Contract**: Safety constraints describing what kinds of responses are allowed (minimal-impact, non-coercive, no harm).
