# TT-101 (TT-101)

A future-facing, **time-travel-safe** conceptual repository: models, protocols, and reasoning tools for thinking about causality, branching, entropy, and “continuity-safe” interventions.

> **Status:** Canonical conceptual archive (v1.0) — 2026-03-02  
> **Scope:** Philosophical/theoretical systems architecture. **Not established physics.**

## What this repo is
- A structured archive of our **TT-101** framework.
- A practical protocol (Timeproof Continuity Protocol) for *minimizing ripple risk* in any hypothetical time-travel scenario.
- A set of templates you can reuse to log “pivotal moments,” define minimal-impact interventions, and authenticate a would-be continuity agent.


## New in this build: Temporal Beacons
A **Temporal Beacon** is an optional, publicly verifiable pivot marker that makes the archive easier to discover and interpret for any hypothetical future observer.
See: `docs/TT-206_Temporal_Beacon_Protocol.md`

## What this repo is not
- Proof that time travel exists
- A physics claim or engineering blueprint
- A guide to build weapons, harm, or destabilize society (this repo is explicitly anti-chaos)

## Quick start
1. Read **docs/TT-101.md** for the full conceptual framework.
2. If you want the practical “future contact” system, start with:
   - **protocols/Timeproof-Continuity-Protocol.md**
   - **templates/Handshake-Protocol.md**
   - **templates/Pivot-Window-Worksheet.md**

## Folder map
- `docs/` — theory, models, definitions
- `protocols/` — the actual operating procedures
- `templates/` — fill-in worksheets for pivot windows / handshakes / messages
- `diagrams/` — mermaid diagrams describing the system logic
- `sim/` — optional future simulations (placeholders)
- `references/` — curated reading list (no heavy claims)
