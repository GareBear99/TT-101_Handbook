# 3-Part Handshake (Template)

**Goal:** Verify a “future contact” claim without relying on guessable secrets.

## Part A — Private Phrase Seed
- Create a phrase that is never spoken aloud.
- Store it in sealed form across redundant locations.

## Part B — Artifact Proof
- A physical object you create now containing a structured puzzle / hash ladder.
- Must be verifiable quickly, hard to fake.

## Part C — Behavioral Proof
Answer-set about *decision patterns*, not trivia:
- 5–10 prompts that only a real continuity agent could answer without leading.

## Fail-Closed Rule
If any part fails: **no trust, no action.**
